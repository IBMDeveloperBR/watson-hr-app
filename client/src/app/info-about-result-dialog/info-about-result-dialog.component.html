Como a personalidade é definida?

Usamos seu currículo para determinar sua personalidade, para isso convertemos o pdf em texto e enviamos para o Watson.

Por fim usamos o Watson Personality Insights para determinar sua personalidade.

 Como a área é definida?

Apos isso, usamos os parametros definidos na aba "Modificar Parâmetros", para determinar uma pontuação para sua personalidade com base nas áreas

A área(s) com maior pontuação é a área exibida no resultado.
