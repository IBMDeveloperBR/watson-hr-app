Como a personalidade é definida?

Usamos seu currículo para determinar sua personalidade, para isso convertemos o pdf em texto e enviamos para o análise.

Por fim, usamos o Watson Personality Insights para determinar sua personalidade.

IMPORTANTE: Esta aplicação não armazena nenhum dado pessoal do usuário.

 Como a área é definida?

Usamos os parâmetros definidos na aba "Modificar Parâmetros", para determinar uma pontuação com base nas áreas e sua personalidade.

A área(s) com maior pontuação é definida como sua ára de vocação.
